## Rollback boundary

The enhanced profile does not migrate the Asterisk wire rate or rewrite existing
Agent assignments. Reassign an Agent to `telephony_ulaw_8k` to restore the
compatibility downsampling path. Provider-specific diagnostic overrides and
diagnostic tap/recording changes used during investigation were removed before
the release freeze.
