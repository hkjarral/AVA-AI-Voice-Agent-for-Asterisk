## Live-call regression gates on voiprnd

Run at least one selected-scope transfer and one denied-scope attempt on both AudioSocket
and ExternalMedia where configured. Also run allowed and denied Google Calendar, Microsoft
Calendar, and voicemail calls, including one stale or selected-empty fail-closed case. Confirm
provider schema, spoken guidance, execution result, canonical tool names in Call History,
audit generation, cleanup, and post-call hooks all agree. Then run the configured
provider/pipeline golden baseline set to exclude greeting, barge-in, farewell, and transfer
regressions. Archive call evidence before marking these rows PASS.

The live gates intentionally cover the only behavior that cannot be proven without calls in
progress: generation retention across a reload and end-to-end allowed/denied resource behavior.
