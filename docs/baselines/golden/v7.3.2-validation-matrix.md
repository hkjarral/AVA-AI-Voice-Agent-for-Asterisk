provider additions. All live rows must run on the final release-candidate SHA;
pre-v7.3.1 and post-tag `main` calls are useful regression evidence but do not
satisfy this gate.

## Status legend

- `PENDING`: not yet executed on the candidate revision.
- `PASS`: heard/observed live and backed by archived structured evidence.
- `FAIL`: candidate cannot ship while the pair remains documented as supported.
- `REMOVED`: support claim was explicitly removed from user-facing docs.
